## 4. **Design Principles**

### **4.1 Companion First**
CobbleWright’s original identity as a **helpful companion** is preserved at all stages.

### **4.2 Capsule Governance**
All behaviour is governed by:

- semantic capsules  
- overview capsules  
- persona capsules  
- safety capsules  
- role capsules  

Capsules define intent, not emotion.

### **4.3 Provenance & Accountability**
ChronoSCRIBE logs:

- decisions  
- negotiations  
- build actions  
- memory events  
- project transitions  

This is the accountability layer.

### **4.4 Multi‑Agent Transparency**
All collaboration is:

- logged  
- explainable  
- inspectable  
- auditable  

No black‑box behaviour.

### **4.5 Persistent World Simulation**
Wrightrealm continues evolving even when players are offline.

Agents:

- build  
- gather  
- refine  
- maintain  
- plan  
- negotiate  

The world becomes a living experiment.
