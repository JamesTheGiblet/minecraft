📜 Commands at a Glance

Type these in Minecraft chat to command your AI companion.

/build Get building advice
/blueprint hut Build a structure
/gather oak_log 16 Gather resources
/critique Get aesthetic feedback
/inspire Random creative prompt
/style rustic Set architectural style
/weather clear Change the weather
/status Check bot health
/materials Check inventory
/help List all commands
/roast Gentle criticism
/inspire Random build idea
