# Wrightrealm Development Server

This directory contains the dedicated Minecraft server environment for developing and testing The Architectum in the world of Wrightrealm.

## Configuration

- **Version:** 1.21.1
- **Online Mode:** `online-mode=false` (in `server.properties`) to allow the bot to connect locally.
- **Startup Script:** Run `start.bat` to launch the server with optimized Java flags.

The `world` and `logs` directories are intentionally excluded from version control via the root `.gitignore` file.
